# 02 - Input & Output

## Objective
Learn how to take input from the user and display output using C++.

## Concepts Covered
- Output using `cout`
- Input using `cin`
- Reading full lines using `getline()`
- Escape sequences (`\n`, `\t`)
- `cin.ignore()`
- Difference between `cin` and `getline()`
